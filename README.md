# Getting Started

## Requirements:

```
Hibernate: 6.6.1.Final
H2 Database: 2.3.232
Maven: 3.9+
JDK: 17
```

### Clone this repository:

```bash
git clone https://github.com/deepaksorthiya/hibernate-6-orm-getting-start.git
```

```bash
cd hibernate-6-orm-getting-start
```

### Run Test(Maven Should Be Installed):

```bash
mvn clean test
```
